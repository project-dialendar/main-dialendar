## 기획 의도

◼️ 사진 한 장과 한 문장으로 작성하는 오늘의 일력.

◼️ 소소한 하루를 모아 소중한 한 달을 채워보세요! 

◼️ 많은 글을 적지 않고도 감성을 담고 싶은 분들을 위한 일기 서비스

◼️ 태블릿 기기에서 가장 최적화된 UI입니다.


<br>

## ⛏️ 주 사용 기술

◼️ Android Studio

◼️ Java

◼️ Firebase

◼️ Room database
